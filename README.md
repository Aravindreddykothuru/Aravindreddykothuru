## 📁 Featured Projects

<details>
<summary><b>🚀 AttendNet: Multi-Face Recognition Attendance System</b></summary>
<br>

A real-time webcam-based attendance management system replacing traditional manual methods. It extracts 128-D VGG-Face embeddings and performs cosine similarity matching to accurately verify classroom attendance.

| Attribute | Metrics & Specifications |
| :--- | :--- |
| **Stack** | Python, OpenCV, VGG-Face (Transfer Learning), face_recognition (dlib), Flask |
| **Scale** | Custom self-collected dataset containing 30–50 images per person |
| **Performance** | ~97% recognition accuracy evaluated via Precision, Recall & F1-Score |
| **Security** | Real-time webcam system replacing manual classroom attendance; VGG-Face 128-D embeddings + cosine similarity |
| **Impact** | Fully automates class attendance tracking, eliminating proxy logins and manual paperwork |
| **Repository** | [View Codebase](https://github.com/Aravindreddykothuru/aTTENDNET) |

### 🛠️ Professional Architecture Overview
AttendNet utilizes an optimized multi-threaded Python backend to capture video frame buffers. It isolates face regions via dlib's CNN detector, crops and aligns them, and feeds them through a pre-trained VGG-Face model to extract facial signatures. These signatures are compared against a pre-registered database using vectorized cosine similarity.
</details>

<details>
<summary><b>🌾 FarmersRent: Equipment Rental Platform</b></summary>
<br>

A peer-to-peer rental marketplace designed for farmers to list, book, and review agricultural equipment. Features secure authentication, full-text database search, and real-time synchronization.

| Attribute | Metrics & Specifications |
| :--- | :--- |
| **Stack** | Node.js (Express.js), PostgreSQL, Prisma ORM, Supabase (Auth · Storage · Realtime), RESTful API |
| **Scale** | P2P rental marketplace for farmers — listing, booking & reviews |
| **Performance** | Full-text search using PostgreSQL tsvector/tsquery; real-time booking via Realtime WebSockets |
| **Security** | Supabase Auth + JWT + PostgreSQL Row-Level Security (RLS) |
| **Impact** | Supabase Storage for equipment images; enables peer-to-peer equipment sharing |
| **Repository** | [View Codebase](https://github.com/Aravindreddykothuru/farmers) |

### 🛠️ Professional Architecture Overview
The backend is powered by Node.js and Express.js, utilizing Prisma ORM for type-safe database queries against a PostgreSQL instance hosted on Supabase. Images are stored securely via Supabase Storage, and real-time booking coordination is established using WebSockets.
</details>
